# 複数シートのデータを自動集計するVBAマクロ

> **ポートフォリオ用サンプル**
> 用途: 店舗別売上の月次集計
> 言語: VBA（Excel）
> 規模: 約60行（コメント込み）

スクリーンショット入りのPDFマニュアル付きで納品しています。
- **PDF版**: [operation-manual.pdf](operation-manual.pdf)

## 概要

複数のシート（店舗別データ）を「集計」シートに自動転記するExcelマクロです。
「集計実行」ボタンを押すだけで、全店舗のデータが1つのシートにまとまります。

---

## Before / After

### Before（現状の業務フロー）

- 店舗担当者が各シートにデータを入力
- 月末、本社担当者が手作業で1つずつコピペして集計
- **所要時間: 30〜60分／月**
- コピーミス・転記漏れのリスク

### After（マクロ導入後）

- 「集計実行」ボタンを押すだけ
- 全店舗のデータが自動で集計シートに転記
- **所要時間: 数秒／月**
- ヒューマンエラーなし

---

## 想定ユースケース

- **多店舗運営の小売店・飲食店** が月次の本社報告用に売上を集計する
- **複数営業所** の日報を本社で取りまとめる
- **複数チーム** の進捗データを月末にまとめる
- **複数支店** の在庫データを集計する

シナリオは「日付・商品名・数量・売上」になっていますが、項目はカスタマイズ可能です。

## 動作環境

- Microsoft Excel 2013 以降（Windows / Mac）
- マクロ有効ブック（.xlsm）として保存する必要あり

---

## 使い方の概要

1. Excel ファイル（.xlsm）を新規作成
2. 「東京店」「大阪店」「福岡店」「集計」の4シートを準備
3. 各店舗シートに見出しとデータを入力（サンプルデータあり）
4. VBA エディタを開いて `macro.bas` のコードを貼り付け
5. ワークシートに「集計実行」ボタンを設置 → マクロを割り当て
6. ボタンを押すと、全店舗のデータが集計シートに転記される

詳細は `setup-and-usage.md` を参照してください。

---

## カスタマイズ可能なポイント

- **対象シート名**: コード内の `sheetNames` 配列を編集
- **データ項目**: コード内の転記ロジックを編集（現状は4項目）
- **集計先シート名**: 現状「集計」固定 → 変更可

---

## 制限事項

- 各店舗シートのフォーマットが揃っている必要があります
- 1シートあたり10,000行まで対応（必要に応じて拡張可能）
- 数式が入っているセルは「値」として転記されます
