# 顧客リストクレンジングマクロ（Excel VBA）

> **ポートフォリオ用サンプル**
> 用途: 顧客リスト・名簿の重複削除と表記揺れ統一
> 言語: VBA（Excel）
> 規模: 約120行（コメント込み）
> 対応OS: **Windows / Mac 両対応** ✅

---

## 🌐 Windows / Mac 両対応

本マクロは **Windows / Mac いずれの Excel でも動作確認済み** です。

VBAは Mac版での互換性が課題になりがちですが、本マクロは以下の工夫で両対応を実現しています：

| Windows専用機能（避けた） | 両対応の代替手段（採用） |
|---|---|
| `Scripting.Dictionary` | `Collection` で重複判定 |
| `Mid` + `AscW` での文字判定 | `WorksheetFunction.Asc` で全角→半角変換 |
| 標準の `Trim`（半角空白のみ） | `StrongTrim` 自作関数（全角空白・NBSP対応） |

→ **クライアントのOS環境を問わず納品可能**。

## 概要

顧客リスト・名簿などのデータを、**ボタン1つで自動的にクレンジング** する Excel マクロです。重複削除・表記揺れ統一・全角半角変換などをまとめて実行します。

---

## Before / After

### Before（汚いリスト）

```
ID, 会社名,             担当者名,   メールアドレス,           電話番号
1,  株式会社サンプル,   田中 太郎,  tanaka@example.com,       03-1234-5678
2, (株)テスト商事,      鈴木 花子,   suzuki@test.com,         03-2345-6789
3, ㈱マーケット,        高橋 一郎,  takahashi@market.com  ,   ０３－３４５６－７８９０
4,  株式会社サンプル ,   田中 太郎  , tanaka@example.com,       03-1234-5678  ← 重複
5, (有)エンジニアリング, 佐藤 健,    sato@eng.com,             03-4567-8901
6, ,,,,                                                         ← 空行
7, ㈲システム,          山田 美咲,  yamada@system.co.jp ,     03-5678-9012
```

問題点：
- 重複行が混在
- 余分な空白
- 「(株)」「㈱」「株式会社」の表記揺れ
- 全角の電話番号
- 空白行

### After（クレンジング後）

```
ID, 会社名,                  担当者名,  メールアドレス,         電話番号
1,  株式会社サンプル,        田中 太郎,  tanaka@example.com,    03-1234-5678
2,  株式会社テスト商事,      鈴木 花子,  suzuki@test.com,       03-2345-6789
3,  株式会社マーケット,      高橋 一郎,  takahashi@market.com,  03-3456-7890
5,  有限会社エンジニアリング, 佐藤 健,    sato@eng.com,         03-4567-8901
7,  有限会社システム,        山田 美咲,  yamada@system.co.jp,   03-5678-9012
```

→ **重複1件削除 / 空白行1件削除 / 5件整形** が一瞬で完了。

## 効率化の効果

| 項目 | Before（手作業） | After（マクロ） |
|---|---|---|
| 所要時間（1,000件） | 約2〜4時間 | **数秒** |
| 重複の見落とし | 発生しやすい | なし |
| 表記統一の精度 | 人によって差 | 100%統一 |

---

## マクロの機能（5つ）

| # | 機能 | 詳細 |
|---|---|---|
| 1 | 重複行の削除 | メールアドレス基準で重複判定（大文字小文字無視） |
| 2 | 前後の空白除去 | 各セルから不要な空白を削除 |
| 3 | 会社名の表記統一 | (株) / ㈱ / 半角全角を「株式会社」に統一<br>(有) / ㈲ も「有限会社」に統一 |
| 4 | 電話番号の半角化 | 全角数字・全角ハイフン → 半角 |
| 5 | 空白行の削除 | すべてのセルが空の行を削除 |

実行後、メッセージで「整形件数 / 重複削除 / 空白行削除」の3つの数値が表示されます。

## カスタマイズ可能なポイント

- **重複判定キー**: コード内でメールアドレスから別の列に変更可
- **表記統一ルール**: NormalizeCompany 関数に追加可
- **対象列**: 1〜5列以外の構成にも対応可能

---

## 制限事項

- 100,000行までは問題なく動作（10万件以上は要検証）
- 数式が入っているセルは「値」として処理されます
